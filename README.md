# 생활관에서 깨어났더니 뱀서 영웅!

![생활관에서 깨어났더니 뱀서 영웅!](https://github.com/user-attachments/assets/b5645222-3fa6-470d-91c2-01362b8d4cf9)

## 프로젝트 소개

이 프로젝트는 이세계에 떨어진 배병장이 세계를 구하는 

로그라이트 슈팅 액션 게임 입니다. 

주인공 배병장은 수많은 적들을 피하고 싸우며 성장합니다.

* * *

장르: 로그라이트 슈팅 액션 게임 (뱀서라이크)

플레이타임: 5분

개발환경: 2022.3.62f2

플랫폼: Windows

프로젝트 기간: 2026/01/28 ~ 2026/02/03

팀: 배정민, 김은진, 서정아, 장지훈, 정우혁

* * *

## 게임 특징

플레이어 캐릭터에게 몰려오는 적들을 피하라.

적들과 싸우며 경험치석을 모아 레벨업을 하고 무기를 업그레이드 하라.

* * *

## 플레이 방법

이동: WASD

선택: Mouse Left Click

## 실행 환경

Unity: 2022.3.62f2

플랫폼: Windows
